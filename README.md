# Service Desk FastAPI Application

## Опис проєкту

Цей проєкт — REST API, реалізований з використанням FastAPI, який дозволяє керувати заявками на ремонт.
Система підтримує ролі користувачів (User) та адміністраторів (Admin) з різними правами доступу.

## Функціональність
 Користувач (User)

 Створення заявки на ремонт

 Оновлення власної заявки

 Видалення власної заявки

 Перегляд своїх заявок

 Адміністратор (Admin)

 Перегляд усіх заявок

 Видалення будь-якої заявки (незалежно від власника)

 Оновлення статусу заявки 


 ## Основні можливості API

Аутентифікація та авторизація

Розподіл прав доступу за ролями

CRUD-операції для заявок на ремонт

Валідація даних за допомогою Pydantic
